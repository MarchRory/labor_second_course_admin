# 课程考评系统后台 - 教师端
基于开源项目 vue-admin-template 二开的项目，已完成全部基本功能并部署到服务器，域名也买了，等备案和学校网信中心对接。

现在`v1.0版本`包括以下功能
- 学生、用户、课程、模板、词汇、词汇类型、学期、目标规划、学生和课程绑定等一众CRUD
- 数据可视化
- Excel导入和导出
- 完整权限体系
- 可编辑div （差不多就是一个迷你富文本编辑器的雏形）
- 教师考评、审核、赋分结果导出
- ...


# 技术栈
- Vue2
- Vuex
- Vue-router@3
- Axios
- Element-UI
- Echarts
